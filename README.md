# Employee Management System
# TestProject
A React-based Employee Management System implementing full CRUD (Create, Read, Update, Delete) operations. Built using React Hooks (`useState`) with a responsive and modern UI for managing employee records.

## Features

* Add new employees
* View employee records
* Update employee details
* Delete employee records
* Dynamic form handling using React Hooks
* Responsive and modern UI design
* State management using `useState`
* Modular styling using CSS

## Tech Stack

### Frontend

* React
* JavaScript (ES6+)
* CSS3
* Vite

## Project Structure

```text
src/
├── App.jsx
├── App.css
├── main.jsx
```

## CRUD Operations

### Create

Add a new employee by filling out the form.

### Read

View all employee records in a structured list.

### Update

Edit employee information and save changes.

### Delete

Remove employee records from the list.
